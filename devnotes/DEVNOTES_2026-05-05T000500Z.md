# DEVNOTES_2026-05-05T000500Z

## 背景

## 本輪決策

### A. 只修 config-time import path

將 `vite-socket-plugin.ts` 與 `socket.shared.ts` 在 dev 啟動鏈上的 `$lib/*` imports 改為相對路徑。

### B. 不擴大到整個 server layer

這不是 shared contract 或 socket flow 的設計問題，而是 config-time module resolution 問題，因此只修最小必要路徑。
